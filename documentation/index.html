1. Installation Guide

Welcome to The SoundBox template! To get started, follow these simple steps:

  1. Unzip the template folder to your local drive.
  2. Open the folder in your favorite code editor (VS Code, Sublime, etc.).
  3. To view the template, open index.html (root) in any modern web browser.
  4. If you're using a local server (like Live Server extension), ensure the root folder is the base directory.
# Sample local server command (if node installed) npx serve .

2. Customization Guide

Colors & Branding

All colors are managed via CSS variables in assets/css/style.css. Modify the :root selector to update your brand colors globally.

:root { --primary: #7c3aed; /* Your primary color */ --secondary: #ec4899; /* Your secondary color */ --bg: #ffffff; /* Page background */ }

Local Fonts

This template uses local fonts for high performance. They are located in assets/fonts/. To change fonts, update the @font-face rules and the --font-primary variable.

Content Replacement

All images are stored in assets/images/. To replace them, simply overwrite the existing files with your own images using the same filenames (recommended format: WebP).

3. Page Structure

The template is organized into 20+ functional pages:

  • Home Pages: index.html (Main), index-2.html (Alternate Artist Showcase)
  • Core Pages: about.html, services.html, shows.html, blog.html, contact.html
  • Dashboards: dashboard-user.html and dashboard-admin.html (Full feature panels)
  • Auth: login.html, signup.html (with social logins)
