# 🚀 Installation Guide

Welcome to The SoundBox template! To get started, follow these simple steps to set up the template locally.

## Prerequisite
- A modern web browser (Chrome, Firefox, Safari, or Edge).
- (Optional) A code editor like VS Code or Sublime Text.

## Setup Steps
1. **Unzip the Files**: Extract the `music-venue-dashboard` folder to your desired location.
2. **Open the Project**: Open the extracted folder in your code editor.
3. **Run Locally**:
   - **Method A (Direct)**: Double-click the `index.html` file in the root directory to open it in your browser.
   - **Method B (Local Server)**: If using VS Code, right-click `index.html` and select **"Open with Live Server"**.
   - **Method C (Command Line)**: Run `npx serve .` inside the project folder.

## Troubleshooting
If colors or icons are missing, ensure the `assets/` folder is correctly placed relative to the HTML files.
